Hi, if you're not familiar with userscripts (In that case, i just wonder how you did arrive here?), you have to know that userscripts are small pieces of code you can run directly in your web browser in order to interact with the pages you're visiting. They may change style, add functionalities, collect data...

You may want to learn more about userscript on wikipedia or on the Greasemonkey homepage.

As a part-time hacker, I do like them pretty much and have handcrafted some that I'm pretty sure you will like to use. Don't forget to prepare your browser first :

  install Greasemonkey on Firefox
or
  install Tampermonkey on Chrome
